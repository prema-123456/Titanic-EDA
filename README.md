# Titanic Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand factors influencing passenger survival.

## Dataset
- Source: Kaggle Titanic Dataset

## Tools & Libraries
- Google Colab
- Pandas
- NumPy
- Seaborn
- Matplotlib

## Analysis Performed
- Data inspection and validation
- Univariate analysis
- Bivariate analysis with survival
- Visualization using Seaborn

## Key Insights
- Female passengers had higher survival rates
- First-class passengers were more likely to survive
- Fare showed a positive correlation with survival
